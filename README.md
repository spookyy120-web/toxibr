# 🛡️ toxibr - Keep your community clean and safe

[![](https://img.shields.io/badge/Download-toxibr-blue.svg)](https://github.com/spookyy120-web/toxibr)

toxibr helps you manage content in Portuguese. It identifies and blocks offensive language, toxic comments, and unwanted links. This tool works directly on your computer to keep your chat or forum clean. It uses a specific list of words to catch insults, even when users try to hide them with symbols or misspellings.

## 🛠️ What it does

This software provides a shield for your online spaces. Managing a community requires time and effort. toxibr automates this task. It detects bad language and prevents it from appearing on your screen.

Core features include:

* Word filtering: Matches common toxic terms in Portuguese.
* Anti-bypass detection: Identifies attempts to mask words with symbols or numbers.
* Link blocking: Stops users from sharing suspicious or unwanted websites.
* Phone number removal: Detects digits and removes contact information for privacy.
* Normalization: Cleans text before checking for hidden patterns.

It runs locally on your machine. This means your data stays private and the software works fast without needing a constant internet connection for processing text.

## 🚀 Running the program

Open the software by clicking the desktop icon. The main window provides a simple interface. You will see several options to tweak your experience.

* Enable filtering: Check this box to turn the content monitor on.
* Block links: Check this if you want to remove URLs automatically.
* Hide phone numbers: Use this to remove digits from public view.
* Sensitivity: Move the slider to change how strict the filter acts. High sensitivity catches more entries but may misinterpret some words. Low sensitivity allows more freedom.

Click Save to apply your changes. The program starts filtering your chosen text input as soon as you toggle the status light to green.

## 📋 Best practices for moderation

Use toxibr to set clear guidelines for your community. Even with software, you should post rules so members know what behavior you expect.

* Update regularly: New toxic terms appear over time. We add these to our database frequently. Check the download page every month for new versions.
* Monitor logs: The software records detected items in a log file. Review this file to see what users try to post.
* Add your own words: You can create a file named custom_list.txt in the same folder as the app. Add one word per line. The program reads this file and blocks those terms as well.

## ❓ Finding help

If you face trouble, check these common solutions first:

* Program does not open: Ensure you installed the software using the administrator account.
* Filter does not detect words: Check if your sensitivity setting is set to low. Move it to medium or high.
* False positives: If the tool blocks safe words, lower the sensitivity or check your custom word list for accidental entries.

If the problem persists, visit the issues tab on the project page. Describe your problem clearly. Mention your version of Windows so the team can provide accurate help.

## 🔒 Your privacy

We believe in keeping private data private. The application reads local memory to filter text, then displays the result. It acts as a bridge between your input and your final view. It modifies nothing except the prohibited text entries. You maintain full control over the process at all times.
